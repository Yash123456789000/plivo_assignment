<h2 style="font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; color: #1a73e8; margin-top: 2rem;">📋 Project Overview</h2>

<p style="font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; color: #333;">
  This is a fullstack application for managing different types of services like websites, APIs, and databases. It supports multi-tenant user access with role-based permissions (admin and user).
</p>

<ul style="font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; color: #333; padding-left: 1.2rem;">
  <li>🌐 <strong>Admins</strong> can log in to view and manage all users and their services.</li>
  <li>👤 <strong>Users</strong> can upload and manage their own services via intuitive dashboards.</li>
  <li>📊 Graphs show monthly usage and pie charts for cost breakdowns.</li>
  <li>🛠 Admins can schedule maintenance or view metrics in detail.</li>
  <li>🌗 Support for light and dark themes, with customizable color themes.</li>
</ul>

<h2 style="font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; color: #1a73e8;">🚀 Features</h2>

<ul style="font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; color: #333; padding-left: 1.2rem;">
  <li>✅ User authentication with role-based access (admin & user)</li>
  <li>✅ CRUD operations for websites, APIs, and databases</li>
  <li>✅ Status tracking: Operational, Degraded, Outage</li>
  <li>✅ Realtime updates using WebSockets</li>
  <li>✅ Public + private dashboards</li>
  <li>✅ Monthly user activity charts</li>
  <li>✅ Cost breakdown pie charts</li>
  <li>✅ Theme switcher: Light/Dark & custom theme colors</li>
</ul>

<h2 style="font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; color: #1a73e8;">🛠 Tech Stack</h2>

<ul style="font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; color: #333; padding-left: 1.2rem;">
  <li><strong>Frontend:</strong> React, Context API, Chart.js, Custom CSS</li>
  <li><strong>Backend:</strong> Node.js, Express</li>
  <li><strong>Database:</strong> MongoDB with Mongoose</li>
  <li><strong>Auth:</strong> Clerk / Auth0</li>
  <li><strong>Hosting:</strong> Vercel (frontend), Render (backend)</li>
</ul>
